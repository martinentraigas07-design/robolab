# RoboLab

## Plataforma educativa para robótica, programación y electrónica

RoboLab es una plataforma educativa desarrollada para acompañar el aprendizaje de robótica mediante desafíos prácticos, simulaciones y el uso de robots reales.

Forma parte del **Ecosistema FRANKY**, un conjunto de herramientas diseñadas para la enseñanza de robótica, programación, electrónica, IoT e inteligencia artificial en instituciones educativas.

---

# Características

* Aprendizaje basado en desafíos.
* Actividades para docentes y estudiantes.
* Integración con FRANKY 4.0.
* Integración con ROBOARD 6.0.
* Recursos didácticos.
* Simulación de actividades.
* Programación visual.
* Preparación para proyectos reales.

---

## Requisitos del sistema

### Sistema operativo

* Windows 10 o superior (64 bits)

### Software necesario

* Python 3.11 o superior (recomendado)
* Navegador web actualizado (Chrome, Edge o Firefox)

### Hardware recomendado

* Procesador Intel Core i3 o superior
* 4 GB de memoria RAM (mínimo)
* Resolución de pantalla 1366 × 768 o superior


---

## Instalación

1. Instalar **Python 3.11** o una versión superior.
2. Descargar la última versión de RoboLab desde la sección **Releases**.
3. Descomprimir el archivo en una carpeta de trabajo.
4. Ejecutar el archivo de inicio de RoboLab siguiendo las instrucciones incluidas.
5. Si es la primera ejecución, instalar las dependencias que el sistema solicite.
